# 🔐 Login and Registration System (C++)

A console-based Login and Registration System built using C++.  
This project demonstrates file handling, authentication logic, and user validation using persistent storage.

---

## 🚀 Features

- User Registration
- Login Authentication
- Duplicate Username Check
- Persistent Data Storage (users.txt)
- Menu-driven Interface
- Simple and Structured Code

---

## 🧠 Concepts Used

- C++ Functions
- File Handling (ifstream, ofstream)
- Conditional Statements
- Loops
- String Handling
- Basic Authentication Logic
